# 🎨 Interactive Color Selector

An interactive and responsive web application built using **HTML, CSS, and JavaScript** that allows users to select colors by clicking numbered circles.

When a user clicks a number from **1 to 20**, the application dynamically changes the **circle color** and the **entire page background color.

## ✨ Features

* 🔢 Numbers displayed from **1 to 20**
* 🎨 **20 unique colors**
* 🖱 Click a number to change the **page background**
* ♻ Previously selected number resets automatically
* 💻 **Responsive layout**
* 🎯 Smooth color transition effects
* 🟢 Interactive hover animations

## ⚙ How It Works

1. The page dynamically generates **20 circular buttons**.
2. Each button represents a **unique color**.
3. When clicked:

   * The selected button changes color.
   * The **page background updates** to the same color.
   * The previously selected button returns to white.
